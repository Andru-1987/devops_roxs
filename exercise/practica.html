  1. Que es un sistema de control de versiones?

    ES UN SISTEMA QUE NOS DA LA POSIBILIDAD DE MANTENER UN REGISTRO/MONITOREO DE LAS MODIFICACIONES QUE SE LE REALIZA A NUESTRO PROYECTO

  2. Cual es la diferencia entre git y github?

    * Git: Es el controlador de versiones

    * Github: Es el repositorio en la nube de nuestro proyecto
